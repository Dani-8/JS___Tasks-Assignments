Core Functions
Targets: Collection
addDoc() Create a new file with a Random ID.
Targets: Document
setDoc() Create or overwrite a file with a Custom ID.
Targets: Collection
getDocs() List all files inside a folder.
Targets: Document
getDoc() Read one specific file by its ID.
Targets: Document
updateDoc() Edit specific text inside a file.
Targets: Document
deleteDoc() Permanently remove a file.
