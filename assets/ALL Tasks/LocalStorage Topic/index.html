Core `LocalStorage` Methods

Five basic rules for saving data in the browser

LocalStorage helps store data in the browser that doesn’t disappear when you refresh or close the tab. It’s useful for saving things like user settings, login info, or data you want to keep between visits.

Methods
Purpose
How it Works
setItem()
To Save Data
Stores a piece of data. Both the `key` (name) and the `value` (data) must be provided as **strings**.
getItem()
To Retrieve Data
Gets the stored value for a given key. If it’s not found, it gives back nothing.
removeItem()
To Delete Specific Data
Deletes a single key/value pair identified by the `key` name.
clear()
Clear Everything
Removes all saved data from localStorage for this site.
key()
Loop through items.
Takes an index number and returns the key name at that position. Used to loop through all stored keys.
