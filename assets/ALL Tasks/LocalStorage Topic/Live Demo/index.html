`LocalStorage` Live Demo

Interact with the five core methods below.

1. `localStorage.setItem(key, value)`

Saves data. Both key and value must be strings.

2. `localStorage.getItem(key)`

Retrieves the value for a specific key. Returns `null` if the key is missing.

3. `removeItem()` & `clear()` (Delete Data)

4. `localStorage.key(index)` (Iteration)

Retrieves the key name by its index (position, starting at 0). Used for looping.
